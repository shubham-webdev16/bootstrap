# TechBlog Pro - Interactive Blog Layout

# Description

TechBlog Pro is a unique, responsive blog website built using Bootstrap 5. It features an interactive hero carousel, dynamic blog post cards with modal popups, a sidebar for navigation, pagination, and a newsletter signup in the footer. This project goes beyond a simple layout by incorporating advanced Bootstrap components for a professional, engaging user experience.

# Features

Responsive Navbar: Includes brand, navigation links, and a search bar.
Hero Carousel: Auto-playing carousel with themed slides for visual appeal.
Blog Posts: Grid of cards with images, titles, descriptions, and "Read More" buttons that open modals.
Sidebar: Categories and recent posts for easy navigation.
Pagination: For browsing multiple pages of posts.
Interactive Modals: Expandable content without leaving the page.
Newsletter Signup: Email subscription form in the footer.
Social Icons: Links to social media platforms.
Hover Effects: Custom animations on cards for interactivity.
Fully Responsive: Adapts to all screen sizes using Bootstrap's grid system.

# Technologies Used

HTML5: Structure of the webpage.
CSS3: Custom styles for animations and theming.
Bootstrap 5: Framework for responsive design, components (navbar, carousel, cards, modals, etc.).
Bootstrap Icons: For social media and UI icons.
JavaScript: Included via Bootstrap CDN for interactive elements.

# Installation and Setup

Clone or Download: Download the project files or copy the HTML code into a file named index.html.
Open in Browser: Simply open index.html in any modern web browser (e.g., Chrome, Firefox).
No server required; it's a static HTML page.
Customize: Edit the HTML to change content, images (replace placeholder URLs), or styles.

# Usage

Navigate the site using the navbar links.
Browse blog posts in the main content area.
Click "Read More" on any card to view expanded content in a modal.
Use the sidebar to filter by categories or view recent posts.
Subscribe to the newsletter via the footer form.
Test responsiveness by resizing your browser window or using developer tools.
